
# Zouq Store Odoo Theme Module

هذا المجلد يحتوي على ملفات ثيم "ذوق" المخصص لنظام Odoo 16/17.

## طريقة التثبيت:
1. قم بتحميل المجلد `odoo_theme_zouq`.
2. انسخ المجلد إلى مسار الـ `addons` الخاص بسيرفر أودو لديك.
3. تأكد من وجود ملف `__manifest__.py` في المسار الصحيح.
4. اذهب إلى إعدادات أودو وفعل "Developer Mode".
5. اذهب إلى قائمة **Apps** واضغط على **Update Apps List**.
6. ابحث عن "Zouq Theme" واضغط على **Install**.
7. بعد التثبيت، اذهب إلى الـ **Website Builder** وستجد "Zouq Hero Banner" متاحاً في قائمة الـ Snippets.

## الملاحظات التقنية:
- تم استخدام SCSS ليتوافق مع نظام أودو الأساسي.
- تم تحويل الـ Layout ليكون RTL بشكل افتراضي ليناسب اللغة العربية.
- الألوان معرفة كمتغيرات SCSS لسهولة التعديل من الـ Theme Customizer الخاص بأودو.
